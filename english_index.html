JavaScript Project: Solo Board Game

You are the Empress’s cartographer, tasked with mapping out the surrounding lands by placing various shaped and terrain elements on an 11x11 grid, all while earning points based on different missions. PHP Project: ELTE IK-Themed Trading Card Game, "IKémon"

The IKémon project is a PHP-based web application where users can trade Pokémon-style cards. In the game, an administrator can create new cards and monitor users' activities. Every user can buy and sell cards, and the cards come with various properties and elements. The goal is for users to collect as many valuable cards as possible.

A screenshot from Fireball Arena gameplay

Java Group Project: Bomberman (Fireball Arena)

In this project, we developed a multiplayer "Bomberman"-style game in Java, where two players compete on a 2D map. Players can place bombs, which explode and remove obstacles and enemies. The project was completed as a team effort, and we managed to implement several bonuses, such as increasing bomb range, as well as monsters that hinder the players. The game also features local multiplayer functionality.
